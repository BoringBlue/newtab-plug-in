# newtab-plug-in（SimpleTab）

>这是一款可以让新建标签页变得看起来简洁文艺的插件
>你可以下载下来自己去修改，做成自己喜欢的样式

- 下载解压
- 打开浏览器->设置->更多工具->扩展程序
- 点击加载以解压的扩展程序，选择该项目的解压目录
